DEV CHALLENGE
===============================

### Risposte 

Nella root del progetto ci sono 4 cartelle contenenti le risposte corrispondenti al numero delle domande.

### Risposta n° 5

Un approccio molto pratico e veloce è ElasticSearch abbinato al plugin Kibana per la parte di storage dei dati e interfaccia di admin. 
Mentre per la parte di backend (in previsione di picchi di utenze simultanee) è consigliabile NodeJS.
Una valida alternativa a NodeJS è il framework di PHP Slim in combinazione con Nginx molto più scalabile di Apache.
